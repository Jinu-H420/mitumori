# 曲げ価格表・見積レスポンス

## 曲げ価格表.csv

| 列 | 意味 |
|:---|:---|
| No. | 形状番号（1=L, 2=コの字, 3=Z, 4=C型, 11=ハット） |
| 形状 | 曲げ形状名 |
| 重量範囲 | 重量クラス上限（kg）。10,20,30,50,80,100,150, 9999=超 |
| 〜835mm 〜 3048mm超 | 長さクラス別の**基準単価（円）** |

**使い方**: 重量と展開長（または製品長）で行を特定し、該当長さ列の単価を参照。必要に応じて数量補正・難易度補正をかける。

---

## 見積レスポンス例.json

APIやスクリプトの見積結果の型の例。

| キー | 意味 |
|:---|:---|
| unit_price | 加工単価（補正後・円） |
| material_cost | 材料費（1個あたり・円） |
| total_price | (単価+材料) × 個数 など総額 |
| details.base_price | CSVから得た基準価格 |
| details.quantity_factor | 数量による補正係数 |
| details.complexity_factor | 難易度補正係数 |
| details.weight_class | 採用した重量クラス |
| details.length_class | 採用した長さクラス |

---

- **例**: `data/見積レスポンス例.json`
